# Hybrid-Financial-Decision-Support-System-Random-Forest-RAG-Gemini-2.5-Flash-
## 📖 Overview

This repository presents the implementation of a **hybrid AI framework for personalized investment advisory**. Accurate and reliable financial recommendations are critical for effective decision-making, minimizing risks, and improving investment outcomes. Traditional financial advisory systems often rely on static data, generic rules, or standalone models, which struggle to provide **personalized, explainable, and up-to-date recommendations**.

To address these challenges, the project integrates three key components—**Random Forest**, **Retrieval-Augmented Generation (RAG)**, and **Large Language Models (LLMs)**. This approach enables the system to automatically analyze investor profiles, retrieve relevant contextual information, and deliver personalized investment recommendations.

Experimental results demonstrate that the proposed **hybrid framework significantly outperforms conventional approaches**, showcasing the effectiveness of combining machine learning, retrieval systems, and generative AI in improving recommendation quality. These findings highlight the potential of building **trustworthy and scalable financial advisory systems**.
